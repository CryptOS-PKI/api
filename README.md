# api

Shared `.proto` definitions and generated gRPC code for [CryptOS-PKI](https://github.com/CryptOS-PKI), consumed by both [`cryptos`](https://github.com/CryptOS-PKI/cryptos) and [`manager`](https://github.com/CryptOS-PKI/manager).

Published as a versioned, standalone Go module. Both consumers import via `go.mod`.

## Status

Pre-alpha. Surface is unstable; tag-based semver kicks in once the v1 contract solidifies.
